Wally IPS

A simple web panel to be feed with data from an IPS (Indoor Positioning System) application and generate some visualizations. This project was tought to receive data with a timestamp, label and user id. It wasn't connect with a real application, so artificial data was generated thinking of it as a Shopping Mall System.

  • Código Fonte: Github.
  • Languages: Python, HTML, JQuery and CSS.
  • Frameworks: Flask.
  • Other technologies: Pandas and MySQL.
